### 第三章 变量和类型

Go语言不支持任何的隐式转换

Go语言支持指针类型，但不支持指针运算

Go语言的字符串默认是空字符串，不是```nil```

### 第四章 运算符

算术、逻辑、比较和位运算符 这些与其他语言基本没有差别

其中```++```和 ```--```都只有后置，为了语言的简单

数组的维数和个数相当，也可以直接进行比较

### 第五章 条件判断语句

if语句支持赋值语句

switch/case支持多个值，不需要写break

switch 也有类似于 if/else的用法
